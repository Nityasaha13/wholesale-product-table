Displays a wholesale product table via a shortcode with configurable columns.

== Description ==
The Wholesale Product Table plugin displays a wholesale product table via a shortcode with configurable columns. It offers AJAX add-to-cart functionality, variable product dropdowns (with updated price and image), pagination, search, and category filtering â€“ making it easy for wholesale customers to browse and purchase products.

== Features ==
* Display wholesale product tables with configurable columns.
* Use a shortcode to embed the table anywhere on your site.
* AJAX add-to-cart functionality for a seamless shopping experience.
* Variable product dropdowns with real-time price and image updates.
* Pagination for easy navigation through large product lists.
* Integrated search and category filtering.
* Full integration with existing WooCommerce products.

== Installation ==
1. Upload the `wholesale-product-table` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Insert the shortcode `[wholesale_product_table]` into any post or page where you want the table to appear.
4. Configure the columns and settings via the plugin settings page in your WordPress admin dashboard.

== Frequently Asked Questions ==
= How do I use the shortcode? =
Simply add `[wholesale_product_table]` to the content area of your post or page.
= Can I customize the columns? =
Yes, you can configure the displayed columns via the plugin settings.
= Does it support variable products? =
Yes, variable products are supported with dynamic dropdowns that update the price and image.
